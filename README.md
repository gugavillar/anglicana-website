# 🌿 Igreja Anglicana Vida

Site institucional oficial da Igreja Anglicana Vida.

Este projeto foi desenvolvido com o propósito de fortalecer a presença digital da igreja, comunicar sua missão e valores, e servir como um ponto de conexão entre a comunidade e aqueles que desejam conhecer mais sobre Cristo.

---

## ✝️ Propósito

Através deste site buscamos:

- Comunicar o evangelho com clareza
- Apresentar nossa missão, visão e valores
- Divulgar eventos e programações
- Facilitar o contato com a igreja
- Expandir o alcance da mensagem de Cristo por meio da tecnologia

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas que garantem performance, escalabilidade e manutenção simplificada:

- **TanStack Start** – Estrutura full-stack moderna baseada em React
- **React** – Biblioteca para construção de interfaces
- **TypeScript** – Tipagem estática para maior segurança no código
- **Tailwind CSS** – Estilização utilitária e design responsivo
- **Biome** – Padronização, lint e formatação automática de código

## 🚀 Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/seu-usuario/anglicana-website.git
cd anglicana-website
```
---

### 2️⃣ Instalar dependências

```bash
pnpm install
```

---

### 3️⃣ Rodar em ambiente de desenvolvimento

```bash
pnpm dev
```
